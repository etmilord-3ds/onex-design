# Modals
Add dialogs to your site for lightboxes, user notifications, or completely custom content.

## Examples

### Static Markup
Below is a static modal dialog (without the positioning) to demonstrate the look and feel of the Modal.

</Canvas>

When backdrop is set to static, the modal will not close when clicking outside it. Click the button below to try it.

    <Story name="StaticBackdrop" story={stories.StaticBackdrop} />

### Without Animation
A Modal can also be without an animation. For that set the ```animation``` prop to ```false```.

    <Story name="WithoutAnimation" story={stories.WithoutAnimation} />

### Vertically centered
You can vertically center a modal by passing the ```centered``` prop.

    <Story name="VerticallyCentered" story={stories.VerticallyCentered} />

### Optional Sizes
You can specify a bootstrap large or small modal by using the ```size``` prop.

    <Story name="Sizes" story={stories.Sizes} />

### Fullscreen Modal
You can use the ```fullscreen``` prop to make the modal fullscreen.
Specifying a breakpoint will only set the modal as fullscreen below the breakpoint size.

<Canvas>
    <Story name="Fullscreen" story={stories.Fullscreen} />
</Canvas>

You can apply custom css to the modal dialog div using the ```dialogClassName``` prop.
Example is using a custom css class with width set to 90%.

### Live demo
A modal with header, body, and set of actions in the footer.
Use ```<Modal/>``` in combination with other components to show or hide your Modal.
The ```<Modal/>``` Component comes with a few convenient "sub components": ```<Modal.Header/>```, ```<Modal.Title/>```,
```<Modal.Body/>```, and ```<Modal.Footer/>```, which you can use to build the Modal content.

<Canvas>
    <Story name="Demo"
           args={{
               animation: false,
               centered: true
           }}>
        {(args) => {
    const [show, setShow] = React.useState(false);
    return (
        <>
            <Button variant="primary" onClick={() => setShow(true)}>
                Launch demo modal
            </Button>
            <Modal {...args} show={show} onHide={() => setShow(false)}>
                <Modal.Header closeButton>
                    <Modal.Title>Modal heading</Modal.Title>
                </Modal.Header>
                <Modal.Body>Woohoo,you are reading this text in a modal!</Modal.Body>
                <Modal.Footer>
                    <Button variant="secondary" onClick={() => setShow(false)}>
                        Close
                    </Button>
                </Modal.Footer>
            </Modal>
        </>
    );
        }}
    </Story>
</Canvas>
