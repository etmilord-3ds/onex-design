# Avatar

An avatar is a graphical representation of a user and typically appears in list and card views,
along side a user's information and/or content that they have created.

## Usage

### Default avatar
<Canvas>
  <Story id="components-avatar--default" />
</Canvas>

### Letter avatar

Avatar containing simple characters can be created by passing a string as ```children```
or by passing ```name``` prop, which used to generate initials of the person.

<Canvas>
  <Story id="components-avatar--initials" />
  <Story id="components-avatar--name" />
</Canvas>

### Image avatar

Image avatar can be created by passing standard img props ```src``` to the component.
<Canvas>
  <Story id="components-avatar--image" />
</Canvas>

### Icon avatar

Icon avatar is created by passing an icon as ```children```.

<Canvas>
  <Story id="components-avatar--icon" />
</Canvas>
