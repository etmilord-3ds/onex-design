# Avatar

An avatar is a graphical representation of a user and typically appears in list and card views,
along side a user's information and/or content that they have created.

## Usage

### Default avatar

<Canvas>
  <Story name="Default" story={stories.Default} />
</Canvas>

### Image avatar

Image avatar can be created by passing standard img props `src` to the component.

<Canvas>
  <Story name="Image" story={stories.Image} />
</Canvas>

### Icon avatar

Icon avatar is created by passing an icon as `children`.

<Canvas>
  <Story name="Icon" story={stories.Icon} />
</Canvas>

### Sizes

An avatar component can vary in size.

<Canvas>
  <Story name="Extra Small" story={stories.ExtraSmall} />
  <Story name="Small" story={stories.Small} />
  <Story name="Medium" story={stories.Medium} />
  <Story name="Large" story={stories.Large} />
</Canvas>
